A sweet and little Wordpress plugin that allows you to link any twitter ID along with the URL of the Twitter profile.


== Installation ==

*   If you are installing locally on your machine then simply copy **twerrific.php** in **wp-content/plugins/** otherwise upload **twerrific.php **in** ****wp-content/plugins/**
*   Go to Admin Panel and select **Plugins** Section and activate **Twerrific Lite** plugin. That\'s it!
== Frequently Asked Questions ==

**Do I still need to make a link to the Twitter profile for any Twitter Id written with @ sign?**

 

Not at all! that\'s the main purpose of the plugin. You just mention twitter Id with \'@\' sign in your post and rest of the work will be done by this plugin
